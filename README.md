# GroundCommands

A simple **Command-to-Marker** script for ESX.  
Allows server commands to be visualized as markers on the ground, with optional keybind functionality.

---

## Installation

1. Drag and drop the `groundcommands` folder into your `resources` directory.  
2. Modify the `client` file as needed for your server setup.  
3. Start the resource in your `server.cfg` (e.g., `ensure groundcommands`).

---

## Features

- Convert server commands into **ground markers**.  
- Optional **keybinds** for certain commands.  
- Easy to set up and customize.
